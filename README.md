#  OOPS Banner App

##  Overview

**OOPS Banner App** is a Java-based console application designed to demonstrate the evolution of programming concepts from basic output to advanced Object-Oriented Design. The project progressively implements multiple use cases (UC1–UC8), showcasing structured development using **GitFlow** and clean coding practices.

---

##  Objectives

* Understand step-by-step transformation from simple printing to scalable design
* Apply **Object-Oriented Programming (OOP)** concepts
* Implement **clean and modular code**
* Practice **version control using GitFlow**

---

## 🧠 Features

### UC1 – Basic Output

* Prints "OOPS" using a simple Java program

### UC2 – Banner Pattern

* Displays "OOPS" using ASCII pattern

### UC3 – Optimization

* Refactors code using `String.join()` for cleaner formatting

### UC4 – Array + Loop

* Stores banner lines in an array and prints using loops

### UC5 – Inline Initialization

* Uses inline array initialization for concise code

### UC6 – Functions

* Separates logic into reusable methods (`getBanner`, `printBanner`)

### UC7 – Class-Based Design

* Introduces `CharacterPattern` class
* Applies encapsulation and object creation

### UC8 – HashMap (Dynamic Rendering)

* Uses `HashMap<Character, String[]>`
* Dynamically prints any string (e.g., "OOPS")
